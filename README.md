# blog_aiohttp
a aiohttp webapp
在aiohttp框架基础上开发的web应用
功能：
1.日志编写，用户评论，管理员后台管理界面功能。
2.仅管理员可以进入后台使用管理员功能需手动将User模型中的admin字段手动设置为1；
